字符串

数组

数学

  • 一、大小写转换

    1 str.toLowerCase() 所有字符转换成小写

    2 str.toUpperCase() 所有字符转换成大写

  • 二、获取指定位置字符

    1 str.charAt(index)

    "abc".charAt(0)==>"a"

  • 三、获取指定位置字符的unicode编码

    1 str.charCodeAt(index)

    "abc".charAt(0)==>97

  • 四、查找关键字

    1 str.indexOf(ele)

    "abc".indexOf("a")==>0

    2 str.lastIndexOf("a")

    "abc".lastIndexOf("a")==>2

  • 五、截取字符串

    1 str.substr(start,len)

    2 str.slice(start,end) 支持负数

    3 str.substring(start,end) 不支持负数

  • 六、字符串变数组

    str.split(reg,count)

    "a|b|c".split("|",2)==>["a","b"]

  • 七、字符串替换

    str.replace(reg,item)

    "abc".replace("a","d")=>"dbc"

  • 八、去除两边空格

    str.trim()

    " abc ".trim()=>"abc"

以上是一些常用的方法,更多的请看菜鸟教程http://www.runoob.com/jsref/jsref-obj-string.html

  • 连接多个数组 concat()

    arr1=["1","2","3"],arr2=["4","5","6"]

    arr1.concat(arr2)=>["1","2","3","4","5","6"]

  • 数组变字符串 join()

    arr1=["1","2","3"]

    arr1.join("++")=>1++2++3

    arr1.join()=>1,2,3

  • pop()

    arr1=["1","2","3"]

    arr1.pop()=>3,arr1=["1","2"]

  • push()

    arr1=["1","2","3"]

    arr1.push("4")=>4(新数组的长度),arr1=["1","2","3","4"]

  • shift()

    arr1=["1","2","3"]

    arr1.shift()=>"1"(删除并返回数组第一个元素),arr1=["2","3"]

  • unshift()

    arr1=["1","2","3"]

    arr1.unshift("4")=>4(新数组的长度),arr1=["4","1","2","3"]

  • reverse()

    arr1=["1","2","3"]

    arr1.reverse()=>(翻转数组),arr1=["3","2","1"]

  • unshift()

    arr1=["1","2","3"]

    arr1.unshift("4")=>4(新数组的长度),arr1=["4","1","2","3"]

  • slice()

    arr1=["1","2","3","4","5"]

    arr1.slice(0,2)=>["1","2"],arr1=["4","1","2","3"]

  • sort()

    arr1=["1","3","5","2","4"]

    arr1.sort()=>(对原数组排序),arr1=["1","2","3","4","5"]

  • splice(index,num,item1,item2,...)

    index:操作位置

    num:操作数量

    添加的元素

    arr=[1,2,3,4,5,6]

    arr.splice(2,2,7,8,9)=>[3,4],arr=[1,2,7,8,9,5,6]

  • every()

    arr=[1,2,3,4,5,6]

    arr.every(v=>{ return v>3 })=>false;

  • map()

    arr=[1,2,3,4,5,6]

    arr.map(v=>{ return v*2 })=>[2,4,6,8,10,12]

  • filter()

    arr=[1,2,3,4,5,6]

    arr.filter(v=>{ return v>3 })=>[4,5,6]

  • forEach()

    arr=[1,2,3,4,5,6]

    arr.forEach(v=>{ console.log(v+1) })=>(2,3,4,5,6,7)

以上是一些常用的方法,更多的请看菜鸟教程http://www.runoob.com/jsref/jsref-obj-array.html

  • Math.min(x,y,...,z)

  • Math.random()

  • Math.round(x)

  • Math.sin(x),Math.cos(x),Math.tan(x)
